# OthelloPC

This is a Java version of the Othello game to run on Windows PC, based on the WiiU version [Othello](https://github.com/MartinButlerAAA/Othello).

The player plays against the computer, alternating who plays first. The aim is to surround and flip the opponents pieces to your colour; the one with the most at the end is the winner.

These are the full [Rules and Instructions for Reversi and Othello](https://www.mastersofgames.com/rules/reversi-othello-rules.htm).

Hope you like it.

This is another A-Level project example, similar to [Connect4](https://github.com/MartinButlerAAA/Connect4).
